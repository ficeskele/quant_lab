# 🚀 快速啟動指南

## 步驟 1: 安裝依賴

```bash
npm install
```

## 步驟 2: 選擇啟動模式

### 🎮 正常模式（體驗完整應用）
```bash
npm run dev
```

### 🐛 調試模式（測試手勢偵測）
```bash
npm run debug
```
這會打開專門的手勢調試工具，顯示：
- 即時的捏合距離和閾值
- 每根手指的伸直/彎曲狀態
- 掌寬數值
- 手勢判定條件

### 🧪 測試模式
```bash
npm run test
```

### 🔍 系統檢查
```bash
npm run check
```

## 步驟 3: 允許攝像頭權限

當瀏覽器彈出攝像頭權限請求時，點擊「允許」。

## 步驟 4: 等待載入

- 等待 2-3 秒讓 MediaPipe 模型載入
- 載入畫面會自動消失
- 你應該會看到視頻畫面和粒子效果

## 步驟 5: 測試手勢

將手放入攝像頭畫面，嘗試以下手勢：

1. **🤏 捏合** - 拇指和食指接觸
   - 可以抓取一團粒子

2. **✌️ 雙指** - 只伸出食指和中指
   - 創造渦旋效果

3. **🖐️ 五指張開** - 完全張開手掌
   - 粒子向外爆開

## 🔍 調試模式

預設已啟用調試模式，你會看到：

- **綠色骨架線** - 顯示手部結構
- **紅點** - 標記重要的關鍵點（手腕、指尖）
- **黃色文字** - 當前偵測到的手勢名稱

**提示**: 如果看不到綠色骨架，表示 MediaPipe 沒有偵測到你的手。

## ❌ 遇到問題？

### 使用測試頁面

訪問測試頁面來診斷問題：

```
http://localhost:3000/test.html
```

測試頁面會顯示：
- ✅ MediaPipe 是否成功初始化
- ✅ 攝像頭是否正常工作
- ✅ 手部是否被偵測到
- ✅ 即時手勢偵測結果

### 常見問題

**問題: 看不到視頻畫面**
- 檢查攝像頭權限
- 確認攝像頭沒有被其他應用占用
- 嘗試重新整理頁面

**問題: 手勢無法偵測**
- 確保光線充足
- 手部完整出現在畫面中
- 避免背景過於複雜
- 查看是否有綠色手部骨架顯示

**問題: 粒子沒有反應**
- 打開瀏覽器控制台（F12）
- 查看是否有錯誤訊息
- 確認手勢名稱顯示是否正確

**問題: 畫面卡頓**
- 降低粒子數量（編輯 `src/main.js`）
- 關閉其他瀏覽器分頁
- 使用較新的電腦或顯卡

### 需要更多幫助？

1. 查看完整的 [README.md](./README.md)
2. 檢查瀏覽器控制台的錯誤訊息
3. 確認網路連線（需載入 MediaPipe 模型）

## 🎯 使用技巧

### 最佳效果

- **光線**: 確保臉部和手部都有良好的照明
- **背景**: 使用單色背景效果最佳
- **距離**: 手部距離攝像頭約 30-50 公分
- **穩定**: 保持手勢穩定至少 0.5 秒

### 手勢技巧

- **捏合抓取**: 拇指和食指輕輕接觸即可
- **渦旋**: V 字手勢，其他手指要彎曲
- **排斥/重置**: 五指完全張開，手掌展開到最大

## 🎮 體驗模式

### 自由模式
- 綠色量子粒子
- 完全自由的力場互動
- 嘗試用捏合吸附、雙指渦旋、張開排斥
